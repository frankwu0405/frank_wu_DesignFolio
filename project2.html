Smart Station – Interactive Trolley Return System

A multi-device system to encourage trolley returns through gamification and clear user feedback.

The Problem

Supermarket trolleys are often left scattered in parking lots, creating hazards and requiring manual collection. The existing return process lacks incentives and clear instructions, leading to low compliance from shoppers.

The Solution

We designed 'Smart Station,' an interactive system that gamifies the trolley return process. It includes a public display, a tablet interface for interaction, and a sensing station that confirms the return. By offering a chance to win a prize, we motivate users, while the streamlined interface makes the process quick and easy.

My Role & Process

  1. Conducted user interviews and mid-to-high fidelity usability testing to identify interaction barriers and motivation triggers.
  2. Prototyped a multi-device experience using Figma, including a public display, tablet interface, and sensing station.
  3. Integrated user feedback to simplify user flow, replace visual effects for better accessibility, and resolve privacy concerns with QR code displays.
  4. Applied gamification (wheel of prize) to encourage trolley return behavior and created a clear feedback loop for users.
  5. Iterated based on testing insights to reduce decision fatigue and minimize site congestion.
